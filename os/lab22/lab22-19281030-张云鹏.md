## Linux源码分析
### 内核源码下载

下载内核最新版本源码

```shell
wget https://cdn.kernel.org/pub/linux/kernel/v5.x/linux-5.18.3.tar.xz
```

### 解压源码
```shell
xz -d linux-5.18.3.tar.xz
tar xvf linux-5.18.3.tar
```
